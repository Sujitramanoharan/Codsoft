
# Credit Card Fraud Detection

##  Problem Statement
Develop a machine learning model to detect fraudulent credit card transactions.

---

##  Approach
- Data loading and preprocessing
- Handling missing values
- Train-test split
- Model training using:
  - Logistic Regression
  - Decision Tree
  - Random Forest



## Exploratory Data Analysis (EDA)
- Fraud vs Legit transaction distribution
- Feature correlation heatmap



##  Features Implemented
- Model comparison
- Classification report (Precision, Recall, F1-score)
- Confusion Matrix visualization
- Feature importance analysis (Random Forest)



## Dataset
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Results
- Random Forest performed better in detecting fraud
- Model handled imbalanced data reasonably


## Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
