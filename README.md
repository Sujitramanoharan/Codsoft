# CODSOFT Machine Learning Projects

This repository contains 3 machine learning projects developed as part of CODSOFT internship tasks.



## Projects Included

### Movie Genre Classification
Predicts movie genres based on plot summaries using NLP techniques.

### Credit Card Fraud Detection
Detects fraudulent transactions using classification models.

### SMS Spam Detection
Classifies SMS messages as spam or legitimate using text processing.



## Techniques Used

- TF-IDF Vectorization
- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)
- Data Preprocessing
- Model Evaluation (Accuracy, Confusion Matrix)
