# SMS Spam Detection

## Problem Statement
Build a machine learning model to classify SMS messages as spam or legitimate (ham).



## Approach
- Data cleaning and preprocessing
- Label encoding (ham = 0, spam = 1)
- Feature extraction using **TF-IDF**
- Train-test split
- Model training using:
  - Naive Bayes
  - Logistic Regression
  - Support Vector Machine (SVM)


## Exploratory Data Analysis (EDA)
- SMS length distribution (histogram)
- Spam vs Ham analysis



## Features Implemented
- TF-IDF vectorization (max_features=5000)
- Model comparison
- Accuracy evaluation
- Confusion Matrix visualization



##  Dataset
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset



##  Results
- Naive Bayes performed well for spam detection
- SVM improved classification accuracy

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, NLP
